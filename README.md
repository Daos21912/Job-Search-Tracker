# Job Search Tracker

A lightweight personal job application tracker built with Streamlit. Log applications, track statuses, filter your list, and keep notes — all stored locally in an Excel file.

This is a personal project, but if you find it useful, feel free to use it.

---

## Getting Started

### 1. Install dependencies

Run the installer to create a local virtual environment and install all required packages:

```
install.bat
```

### 2. Launch the app

```
launch.bat
```

This will start the Streamlit app in your browser. When you're done, closing the terminal will shut it down cleanly.

---

## Features

- Add, edit, and delete job applications
- Track company, role, date applied, status, links, notes, and last updated date
- Filter by company, role, status, or date range
- Data stored locally in `data/applications.xlsx` — nothing leaves your machine

---

## Requirements

- Windows
- Python 3.x (must be on your PATH for the installer to work)

All Python dependencies are listed in `requirements.txt` and installed automatically by `install.bat`.
